# Quanti - A Product-Focused Development Studio

A brutalist, editorial website for a product development studio that builds web apps, smart contracts, and API integrations.

## Tech Stack

- React + Vite
- CSS (custom brutalist design system)
- Deployable to GitHub Pages

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deploying to GitHub Pages

1. Update `vite.config.js` with your repository path
2. Run `npm run deploy`
